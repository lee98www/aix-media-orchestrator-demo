AIX Media Orchestrator — PoC 데모 사용 안내
=============================================

■ 시작하기
  1. 압축을 푼 뒤 폴더 안의 index.html 을 더블클릭하세요. (Chrome 또는 Edge 권장)
  2. 인터넷 연결 상태에서 여는 것을 권장합니다. (화면 글꼴을 웹에서 불러옵니다)
  3. index(화면 목록)에서 각 화면으로 이동할 수 있고, 화면 좌측 메뉴로도 오갈 수 있습니다.

■ 이 데모는
  - 뉴스·광고 AI 제작 SaaS "AIX Media Orchestrator"의 PoC 데모입니다.
  - 실제 서버/AI 호출 없이 동작하는 체험판입니다. 생성 버튼은 단계별 진행을
    시뮬레이션한 뒤 미리 준비된 결과를 보여줍니다.
  - 편집·조작(드래그, 텍스트 수정, 레이아웃 변경 등)은 실제로 동작합니다.

■ 화면 구성 (14장)
  01 메인              — 제작 진입 / 온톨로지 위젯 / 산업 템플릿
  02 기사 생성         — 프롬프트, 보도 이미지, 온톨로지 ON/OFF
  03 기사 편집         — 레이아웃 프리셋 적용, 본문 직접 수정
  04 영상 클립 생성    — 길이(15초)·비율 선택, 스토리보드, 생성 큐
  05 영상 클립 편집    — 타임라인 드래그/삭제, 자막 저작(폰트·크기·색·위치), 재생
  06 협업 대시보드     — 링크 공유/초대, 코멘트, To Do
  07 이미지 광고 생성  — 산업 템플릿, 4가지 시안, 비율
  08 이미지 편집       — 텍스트 레이어 추가, 크롭, 배경 제거, AI 변형 제안
  09 마이페이지        — 프로젝트 폴더, 공유 여부, 사용량, API 키
  10 시스템 구성도     — 전체 아키텍처 도식 (문서용)
  11 자산 라이브러리   — 폴더, 공유 설정, AI 생성 출처
  12 Premiere 플러그인 — 실제 프리미어 안에서의 연동 화면 (클립 전송 체험)
  13 온톨로지 스튜디오 — 브랜드 지식그래프 탐색, 개인화 적용 전/후 비교
  14 온톨로지 제안서   — 활용성 제안 문서 (인쇄/캡처용)

■ 조작 방법
  - 토글/칩/탭        : 클릭하면 상태가 바뀝니다. (예: 온톨로지 ON/OFF, 비율 변경)
  - 생성/적용 버튼     : 클릭 시 진행 단계가 표시된 뒤 완료됩니다. (데모 시뮬레이션)
  - 드래그            : 05 타임라인 클립 순서, 08 텍스트/이미지 레이어, 05 자막 위치
  - 더블클릭          : 텍스트·자막 내용 편집
  - 마우스 휠         : 캔버스/프리뷰/그래프 줌 인·아웃 (03 문서는 Ctrl+휠),
                        더블클릭으로 원래 배율 복귀, 빈 영역 드래그로 이동
  - 코멘트 핀(파란 원) : 클릭하면 협업 코멘트 스레드가 열립니다. (05, 08)
  - 신규 추가          : To Do(06), 새 프로젝트(09), 자산 업로드(11), 자막(05),
                        텍스트 레이어(08), 데이터 소스(13) 등 + 버튼 동작

■ 참고
  - 데모 특성상 변경 내용은 저장되지 않으며, 새로고침하면 초기 상태로 돌아갑니다.
  - 화면 속 브랜드/수치/인물은 모두 데모용 가상 데이터입니다.
